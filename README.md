🌍 Tourism Landing Page

This project is a simple tourism landing page built using HTML and CSS. It presents attractive tourist destinations with images and descriptive content in a clean and organized layout.

🎯 Objective

The goal of this project is to practice semantic HTML structure and basic CSS styling while building a visually appealing web page.

🧠 Problem

How to organize and present tourism information in a clear, structured, and user-friendly way using only HTML and CSS.

🛠️ Solution

The page was structured into sections to improve readability and user experience:

Header with title and introduction
Sections for each tourist destination
Images to enhance visual appeal
Descriptive text for each location
Footer with additional information
🧱 Project Structure
index.html — Main HTML structure
style.css — Page styling and layout
🎓 What I Practiced
Semantic HTML elements
Page structure and organization
CSS styling (colors, spacing, layout)
Basic layout design
🚀 Result
t
A clean and simple landing page that simulates a tourism website, focusing on clarity, structure, and visual presentation.
